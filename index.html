B-1b Knowledge Improver/Game

Expand for more information about these flashcards and how this site works

These flash cards are to help improve knowledge about the B-1b Bomber.

You can use the onscreen buttons for the following:

  • "Flip Card" shows the opposite side of the flash card
  • "Next" allows you to skip a card
  • "Correct" will increase the correct score, remove a value from incorrect, and remove that card from the deck
  • "Incorrect" will increase the score for incorrect and move that card to the bottom of the deck
Shotcut keys (below) are also implemented if you would prefer to use a keyboard.
  • Spacebar = Flip
  • "c" = Correct
  • "i" = Incorrect
  • "f" = Flip
